Product Design

Bunz Inventory

Bunz Inventory is the Airbnb of tool sharing application that allows generous people to make money by sharing their items to anyone.

The goal here was to design a user journey using an exisiting design language and using google’s design sprint as a tool to put myself in a time constraint.

 

Building the touchpoints was essential to this project. By doing so, I was able to paint a picture of how a user would likely use this feature.

Not only did it gave me an understanding of how the user would go about the process, it also gave me a “scaffolding” of building the flow.

 

wireframe

Wireframing is an essential process of building a flow. this allowed me to test out the design in its early stages. read more about in the case study

Validating with Usertesting

Validating the design by user testing is the last stage of this project. this was ultimately the stage where it could be handed off to a developer.
