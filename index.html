Product Design

Bunz Inventory

Bunz Inventory is an independent design exercise with a goal to create a feature from an existing product while putting myself in a time constraint. So what is Bunz Inventory anyway? It is a digital platform where users can share their items to other users.

In a two-week span, I have researched and used techniques from google's design sprints to bring my concept to life. Request a copy of the case study to learn more about this project.

Request Case Study
 
 

Importance of Mapping out Actions

Building the touchpoints was essential to this project. By doing so, I was able to paint a picture of how a user would likely use this feature.

Not only did it gave me an understanding of how the user would go about the process, it also gave me a “scaffolding” of building the flow.

 

Validating with User testing

Validating the design by user testing is the last stage of this project. this was ultimately the stage where it could be handed off to a developer.
